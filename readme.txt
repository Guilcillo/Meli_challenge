
El proyecto consta de un notebook que desarrolla el caso de negocios y una carpeta `./utils/` que contiene tres archivos:

1. **`meli_data_extract.py`**: Define una clase para extraer datos directamente desde la API de Mercado Libre (MELI) y preprocesarlos para facilitar su uso.
2. **`meli_eda.py`**: Contiene varias funciones útiles para limpiar datos y realizar análisis estadístico descriptivo.
3. **`meli_model_2.py`**: Incluye una clase que permite modelar los datos utilizando el algoritmo KMeans.

### **Instrucciones para ejecutar el proyecto:**
1. Copia (o clona) la carpeta completa llamada `MELI`.
2. Abre y ejecuta el notebook en tu entorno de Jupyter Notebook o equivalente.

### **Requisitos:**
Opcionalmente, puedes ejecutar el archivo `requirements.txt` para instalar las librerías necesarias para ejecutar el proyecto. Para hacerlo, usa el siguiente comando en tu terminal:

```bash
pip install -r requirements.txt
```
